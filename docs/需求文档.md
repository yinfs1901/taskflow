---

## 一、项目概述

**项目名称**：TaskFlow
**项目类型**：Electron 桌面应用（Electron 28 + React 18 + Vite 6 + TypeScript）

**核心理念**：一款本地优先的个人任务管理工具，强调快速记录、结构化管理和可视化筛选，无需联网，无协作功能。

**技术选型**：

| 层级 | 技术 |
|------|------|
| 桌面框架 | Electron 28.3.3 |
| 前端框架 | React 18 + TypeScript |
| 构建工具 | Vite 6.4 |
| 状态管理 | Zustand |
| UI 组件 | shadcn/ui（基于 Radix + Tailwind） |
| 数据库 | SQLite（better-sqlite3 native binding） |
| 样式主题 | Catppuccin Mocha |

**数据存储路径**：`%APPDATA%/taskflow/taskflow.db`

---

## 二、用户角色与使用场景

- **目标用户**：个人用户（当前阶段），未来可扩展至小型团队协作
- **核心场景**：日常待办记录、项目任务拆解、截止日期跟踪

## 三、数据模型

### 3.1 任务（Task）

| 字段 | 类型 | 说明 |
|------|------|------|
| id | TEXT (UUID) | 主键 |
| title | TEXT | 任务标题（必填） |
| description | TEXT | 任务描述（选填） |
| status | TEXT | 状态：todo / in_progress / done / cancelled |
| priority | TEXT | 优先级：low / medium / high / urgent |
| deadline | TEXT | 截止日期（ISO 8601，nullable） |
| category_id | TEXT | 所属分类（FK，nullable） |
| assignees | TEXT (JSON) | 责任人列表，格式：`[{name, ratio}]`，ratio 为百分比 |
| created_at | TEXT | 创建时间 |
| updated_at | TEXT | 更新时间 |

### 3.2 分类（Category）

| 字段 | 类型 | 说明 |
|------|------|------|
| id | TEXT (UUID) | 主键 |
| name | TEXT | 分类名称 |
| color | TEXT | 颜色（HEX） |
| sort_order | INTEGER | 排序顺序 |

### 3.3 标签（Tag）

| 字段 | 类型 | 说明 |
|------|------|------|
| id | TEXT (UUID) | 主键 |
| name | TEXT | 标签名称 |
| color | TEXT | 颜色（HEX） |

### 3.4 任务-标签关联（Task_Tags）

| 字段 | 类型 |
|------|------|
| task_id | TEXT (FK) |
| tag_id | TEXT (FK) |

## 五、UI/UX 设计

### 5.1 布局结构

```
┌─────────────────────────────────────────────────────────────┐
│  自定义标题栏（拖动区域，窗口控制按钮）                         │
├──────────┬──────────────────────┬──────────────────────────┤
│  侧边栏  │      任务列表         │       任务详情           │
│  (224px) │      (弹性宽度)        │       (360px)            │
│          │                      │                          │
│  搜索框   │  当前筛选标题 + 排序   │  标题（可编辑）           │
│  筛选导航  │  + 任务卡片列表       │  描述（可编辑）           │
│  分类列表  │                      │  状态 / 优先级 / 截止日期  │
│  + 添加   │  [+ 新建任务]        │  分类 / 标签              │
│          │                      │  责任人 + 比例            │
│          │                      │  ──────────────           │
│          │                      │  取消任务 | 删除任务        │
└──────────┴──────────────────────┴──────────────────────────┘
```

### 5.2 视觉风格

- **主题**：Catppuccin Mocha（深色）
- **背景色**：`#1e1e2e`
- **卡片色**：`#181825`
- **边框色**：`#313244`
- **主文字**：`#cdd6f4`
- **次要文字**：`#6c7086`
- **强调色**：按优先级分色（urgent=红 / high=橙 / medium=蓝 / low=灰）

### 5.3 交互规则

- **新建任务**：点击「+ 新建任务」→ 弹框填写 → 提交后在当前筛选列表中显示
- **编辑任务**：点击任务卡片 → 右侧详情面板 → 直接修改字段 → 自动保存（调用 updateTask）
- **详情页更新后不消失**：updateTask 后通过 `task:list { id }` 重新获取最新数据写入 `selectedTask`，确保详情页始终显示当前数据
- **删除任务**：详情页「删除」按钮 → 直接物理删除 → 详情面板清空

## 六、IPC 接口（Electron 主进程 ↔ 渲染进程）

| 通道 | 方向 | 说明 |
|------|------|------|
| task:list | invoke | 查任务列表，支持 id/status/category_id/priority/today/important/search/orderBy |
| task:create | invoke | 创建任务，返回新任务 |
| task:update | invoke | 更新任务字段 |
| task:delete | invoke | 删除任务 |
| category:list | invoke | 查所有分类 |
| category:create | invoke | 创建分类 |
| category:update | invoke | 更新分类 |
| category:delete | invoke | 删除分类 |
| tag:list | invoke | 查所有标签 |
| tag:create | invoke | 创建标签 |
| tag:delete | invoke | 删除标签 |
